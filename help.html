PMX Editor 帮助文档

Web-based PMX Model Editor for MikuMikuDance

快速开始

  1. 点击工具栏 📂 打开文件夹 按钮，选择包含 .pmx 文件的模型文件夹
  2. 编辑器自动找到 PMX 文件并解析，同时加载文件夹中的所有纹理贴图
  3. 3D 视口会显示带纹理的完整模型，左侧面板展示详细数据
  4. 通过左侧选项卡切换不同的编辑面板
  5. 编辑完成后点击 💾 保存 按钮导出修改后的 PMX 文件
支持 PMX 2.0 和 2.1 版本。文件在浏览器本地处理，不会上传到服务器。选择文件夹后会自动匹配并加载纹理（PNG/JPG/BMP/TGA等）。

📋 信息面板

显示和编辑模型的基本信息：

  • 名称 (日文/英文) — 模型在 MMD 中显示的名称
  • 注释 (日文/英文) — 模型的版权信息、制作者说明等
  • 文件统计 — 版本号、编码格式、各数据的数量汇总

📐 顶点面板

查看和编辑模型的顶点数据。

  • 列表默认显示前 200 个顶点，输入索引号可跳转到指定顶点
  • 点击顶点后可编辑：位置 (X/Y/Z)、法线、UV 坐标、边缘缩放

字段说明

  • 位置 — 顶点在 3D 空间中的坐标
  • 法线 — 决定光照效果的方向向量
  • UV — 纹理贴图坐标 (0~1 范围)
  • 边缘缩放 — 铅笔描边的粗细系数，1.0 约为 1 像素

🎨 材质面板

编辑模型的材质属性，控制外观效果。

基本属性

  • 漫反射颜色 (RGBA) — 材质的基础颜色，A 控制透明度
  • 高光颜色 — 反射光的颜色
  • 高光强度 — 高光范围大小。丝绸≈80、皮革≈35、棉布≈5
  • 环境光颜色 — 阴影区域的颜色
  • 边缘颜色/大小 — 铅笔描边效果的颜色和粗细

纹理贴图

  • 加载纹理 — 点击 📁 按钮从本地选择图片文件 (PNG/JPG/BMP/TGA)
  • 纹理预览 — 面板中显示缩略图，同时 3D 视口实时更新
  • 移除纹理 — 点击 🗑️ 按钮清除纹理，恢复纯色显示
  • 纹理路径 — 显示 PMX 文件中记录的纹理文件路径

材质预设

一键应用预设的布料/材质参数：

预设效果适用场景
丝绸强高光、冷调环境光、双面礼服、丝带、内衣
牛仔低光泽、蓝调牛仔裤、牛仔外套
皮革中等光泽、暖色高光皮衣、皮靴、皮带
棉布几乎无高光、柔和T恤、卫衣、休闲裤
天鹅绒轻微光泽、紫调礼服、窗帘
金属极强反射盔甲、首饰、金属件
预设只修改高光、环境光和标志位，不会改变漫反射颜色和纹理。可以先设置颜色/纹理，再应用预设。

标志位

  • 双面 — 关闭背面剔除，薄布料和头发常需要
  • 地面阴影 — 在地面投射阴影
  • 投射阴影 — 向其他物体投射阴影
  • 接受阴影 — 接受其他物体的阴影
  • 边缘 — 启用铅笔描边效果

🦴 骨骼面板

查看和编辑骨骼数据。

  • 名称 — 骨骼显示名（日文为 MMD 主显示，英文为兼容名）
  • 位置 — 骨骼在 3D 空间的初始坐标
  • 父骨骼索引 — 指向父骨骼的编号，-1 表示根骨骼
  • 层级 — 变形计算顺序的层级

骨骼标志

  • 索引尾部 — 尾端连接方式：勾选=连接到另一骨骼，不勾选=偏移向量
  • 旋转 — 允许旋转操作
  • 平移 — 允许平移操作
  • 可见 — 在 MMD 中是否显示
  • 启用 — 用户是否可操作此骨骼
  • IK — 是否为 IK 骨骼（逆向运动学）
在 3D 视口中开启骨骼显示（🦴按钮）可以可视化骨骼位置和层级关系。

😊 表情面板

查看和编辑 Morph（表情/变形）数据。

  • 面板类型 — 决定在 MMD 中显示的分类位置
    • 系统保留 — 基础表情（通常为 base）
    • 眉毛 — 眉毛相关
    • 眼睛 — 眨眼、视线等
    • 嘴巴 — 口型、微笑等
    • 其他 — 其他变形
  • 类型 — Morph 的技术类型（顶点/骨骼/UV/材质/组合等）
  • 偏移数据数量 — 该 Morph 影响的顶点/骨骼/材质数

⚙️ 刚体面板

编辑物理刚体参数，控制模型的物理模拟效果。

  • 形状 — 球体、方盒、胶囊（碰撞体形状）
  • 尺寸 — 碰撞体的三维大小
  • 位置 — 碰撞体的空间位置
  • 质量 — 影响物理模拟的重量
  • 物理模式：
    • 跟随骨骼 — 刚体跟随骨骼运动（如身体部位）
    • 物理演算 — 刚体受重力影响（如头发、裙子）
    • 物理+骨骼 — 以骨骼为支点进行物理演算

🔗 关节面板

编辑连接刚体的关节（约束），定义物理运动的范围。

  • 类型 — Spring 6DOF（弹簧六自由度）最常用，其他为 PMX 2.1 扩展类型
  • 位置 — 关节在 3D 空间中的位置

换装/换布料 工作流

典型的"给角色换衣服材质"操作步骤：

  1. 点击 📂 打开文件夹，选择模型所在目录（纹理会自动加载）
  2. 切换到 🎨 材质面板
  3. 在材质列表中找到要修改的服装材质（如"上衣"、"裙子"）
  4. （可选）点击 📁 加载纹理 更换纹理贴图
  5. （可选）使用 材质预设 快速切换质感（如"丝绸"→"牛仔"）
  6. （可选）手动调整漫反射颜色改变色调
  7. 确认 3D 视口效果满意后点击 💾 保存
选中材质时 3D 视口会高亮对应区域，方便确认是哪个部件。

语言切换

点击右上角的 中文 / EN 按钮切换界面语言。设置会自动保存，下次打开时保持。

注意事项

  • 本工具为纯前端应用，所有文件操作在浏览器本地完成
  • 加载的纹理文件仅用于预览，保存时 PMX 文件只记录纹理路径名
  • 大文件（>100MB）解析可能需要较长时间
  • 建议在 Chrome 或 Edge 浏览器中使用以获得最佳性能
  • 修改后的文件可直接在 MikuMikuDance 或桌面版 PMX Editor 中验证

Getting Started

  1. Click the 📂 Open Folder button in the toolbar and select the model folder containing a .pmx file
  2. The editor automatically finds the PMX file, parses it, and loads all textures from the folder
  3. The 3D viewport displays the fully textured model; the left panel shows detailed data
  4. Switch between editing panels using the tabs on the left
  5. Click 💾 Save to export the modified PMX file when done
Supports PMX 2.0 and 2.1. Files are processed locally in your browser — nothing is uploaded. Selecting a folder auto-matches and loads textures (PNG/JPG/BMP/TGA, etc.).

📋 Info Panel

View and edit basic model information:

  • Name (Japanese/English) — Display name shown in MMD
  • Comment (Japanese/English) — Copyright, author info, usage terms
  • File Statistics — Version, encoding, and data count summary

📐 Vertices Panel

View and edit vertex data.

  • The list shows the first 200 vertices by default. Enter an index to jump to a specific vertex
  • Click a vertex to edit: Position (X/Y/Z), Normal, UV coordinates, Edge scale

Field Reference

  • Position — 3D coordinates in model space
  • Normal — Direction vector for lighting calculations
  • UV — Texture mapping coordinates (0~1 range)
  • Edge Scale — Pencil outline thickness multiplier (1.0 ≈ 1 pixel)

🎨 Materials Panel

Edit material properties that control the model's appearance.

Basic Properties

  • Diffuse Color (RGBA) — Base color; A controls transparency
  • Specular Color — Color of reflected highlights
  • Specular Strength — Size of highlight. Silk≈80, Leather≈35, Cotton≈5
  • Ambient Color — Color in shadowed areas
  • Edge Color/Size — Pencil outline color and thickness

Texture Management

  • Load Texture — Click 📁 to select a local image file (PNG/JPG/BMP/TGA)
  • Texture Preview — Shows thumbnail in panel; 3D viewport updates in real-time
  • Remove Texture — Click 🗑️ to clear texture and revert to solid color
  • Texture Path — Shows the texture filename stored in the PMX file

Material Presets

One-click fabric/material parameter presets:

PresetEffectUse Cases
SilkHigh specular, cool ambient, double-sidedGowns, ribbons, lingerie
DenimLow gloss, blue toneJeans, denim jackets
LeatherMedium gloss, warm specularJackets, boots, belts
CottonNear-zero specular, softT-shirts, hoodies, pants
VelvetSlight sheen, purple toneGowns, curtains
MetalMaximum reflectionArmor, jewelry, hardware
Presets only change specular, ambient, and flags. They don't modify the diffuse color or texture. Apply color/texture first, then a preset.

Flags

  • Double-sided — Disables back-face culling (needed for thin fabrics and hair)
  • Ground Shadow — Casts shadow on the ground plane
  • Cast Shadow — Casts shadow onto other objects
  • Receive Shadow — Receives shadows from other objects
  • Edge — Enables pencil-outline effect

🦴 Bones Panel

View and edit bone/skeleton data.

  • Name — Display name (Japanese is primary in MMD, English for compatibility)
  • Position — Initial position in 3D space
  • Parent Index — Index of parent bone (-1 = root bone)
  • Layer — Deformation calculation order

Bone Flags

  • Indexed Tail — Tail connects to another bone (checked) or uses offset vector (unchecked)
  • Rotatable — Allows rotation
  • Translatable — Allows translation/movement
  • Visible — Displayed in MMD
  • Enabled — User-controllable
  • IK — Inverse Kinematics bone
Enable bone visualization in the viewport (🦴 button) to see bone positions and hierarchy.

😊 Morphs Panel

View and edit morph (facial expression/deformation) data.

  • Panel Type — Category display in MMD:
    • System — Base morph
    • Eyebrow — Eyebrow expressions
    • Eye — Blink, gaze, etc.
    • Mouth — Lip sync, smile, etc.
    • Other — Other deformations
  • Type — Technical morph type (Vertex/Bone/UV/Material/Group)
  • Offset Count — Number of affected vertices/bones/materials

⚙️ Rigidbodies Panel

Edit physics rigid body parameters for simulation.

  • Shape — Sphere, Box, or Capsule (collision shape)
  • Size — Collision body dimensions
  • Position — Collision body location
  • Mass — Weight for physics simulation
  • Physics Mode:
    • Follow Bone — Rigid body follows bone (body parts)
    • Physics — Gravity-affected (hair, skirts)
    • Physics + Bone — Physics with bone as pivot

🔗 Joints Panel

Edit joints (constraints) that connect rigid bodies and define motion ranges.

  • Type — Spring 6DOF is most common; others are PMX 2.1 extensions
  • Position — Joint location in 3D space

Outfit/Fabric Change Workflow

Typical steps to change a character's clothing material:

  1. Click 📂 Open Folder and select the model directory (textures load automatically)
  2. Switch to the 🎨 Materials panel
  3. Find the clothing material in the list (e.g., "jacket", "skirt")
  4. (Optional) Click 📁 Load Texture to replace the texture
  5. (Optional) Use a Material Preset to quickly switch fabric type (e.g., "Silk" → "Denim")
  6. (Optional) Manually adjust the diffuse color to change the hue
  7. Confirm the 3D preview looks correct, then click 💾 Save
When a material is selected, the 3D viewport highlights the corresponding area to help identify which part it is.

Language Switching

Click the 中文 / EN buttons in the top-right corner to switch the interface language. Your preference is saved automatically.

Notes

  • This is a purely client-side application — all file operations happen locally in your browser
  • Loaded textures are for preview only; the PMX file only stores the texture filename
  • Large files (>100MB) may take longer to parse
  • Chrome or Edge browsers recommended for best performance
  • Saved files can be verified in MikuMikuDance or desktop PMX Editor
